Blogue

A group project to build an community journalling webpage using a RESTful API and the express.js library. Users can anonymously post entries, with an optional accompanying GIF of their choice. Other users may view, comment and react on the submitted posts.

Rock, Paper, Scissors

A simple implementation of the classic rock, paper, scissors game. The game is played against the computer and the first to score five points wins the session. Sessions can be restarted at any time. This was a solo project built using HTML, CSS and vanilla JavaScript.

Hobbit

Habit tracker that allows users to log and monitor their habits. Hobbit was built as part of a group project using technologies such as Node and PostgreSQL. JWTs were used for authentication and authorisation as well as Docker for containerisation.
